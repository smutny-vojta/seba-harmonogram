# seba-harmonogram

Czech summer-camp schedule management PWA built with Next.js, TypeScript, MongoDB, Better Auth, and Bun.

## Quick Start

1. Install dependencies:

```bash
bun install
```

2. Start local database:

```bash
bun run db:start
```

3. Run development server:

```bash
bun run dev
```

## Useful Commands

- `bun run build`
- `bun run start`
- `bun run lint`
- `bun run format`
- `bun run db:stop`
- `bun run db:reset`
- `bun run db:seed`
