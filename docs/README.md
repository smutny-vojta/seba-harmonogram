# Projektová Dokumentace

Tento soubor je vstupní rozcestník dokumentace.

Podrobná pravidla jsou v dokumentaci níže.

## O projektu

- `seba-harmonogram` je PWA pro správu táborového harmonogramu.
- Stack: Next.js App Router, React, TypeScript, MongoDB, Better Auth, Tailwind v4, shadcn/ui.
- Runtime a package manager: Bun.
- Lokalizace: čeština.
